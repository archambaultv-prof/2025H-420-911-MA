# Périphériques d'entrée et sortie

Un périphérique est un composant matériel connecté à un ordinateur, permettant
de lui fournir des données (entrée), de recevoir des données traitées (sortie),
ou les deux (entrée-sortie).

Voici quelques exemples:

| Type de périphérique | Exemples |
|---------------------|----------|
| Périphériques d'entrée | Clavier, souris, scanner, microphone. |
| Périphériques de sortie | Écran, imprimante, haut-parleur. |
| Périphériques d'entrée-sortie | Disque dur, clé USB, carte graphique. |

## Interne ou externe

Les périphériques peuvent être internes ou externes. Les périphériques internes
sont intégrés à l'ordinateur, comme le disque dur ou la carte graphique. Les
périphériques externes sont connectés à l'ordinateur via des ports, comme les
claviers, souris et imprimantes. Ils peuvent être connectés par des câbles
ou sans fil (Bluetooth, Wi-Fi).

| Interne ou externe | Exemples |
|-------------------|----------|
| Interne | Disque dur, carte graphique, mémoire vive. |
| Externe | Clavier, souris, imprimante, haut-parleur, imprimante. |

## Ce qui n'est pas un périphérique

Il peut sembler étrange que les disques durs et la carte graphique soient
considérés comme des périphériques. Mais en réalité ils ne sont pas *strictement*
essentiels au fonctionnement de l'ordinateur. Par exemple un serveur se débrouille
très bien sans carte graphique. Les [Raspberry Pi](https://www.raspberrypi.com/) n'ont pas de disque dur
et fonctionnent très bien avec une simple carte SD.

En gros les composants internes qui sont essentiels au fonctionnement de l'ordinateur
sont les suivants:

| Composant | Description |
|----------|-------------|
| Processeur | Unité centrale de traitement ([CPU](./acronymes.mdx)) qui exécute les instructions. |
| Mémoire vive | Mémoire temporaire ([RAM](./acronymes.mdx)) utilisée pour stocker les données en cours d'utilisation. |
| Carte mère | Circuit imprimé principal qui connecte tous les composants. |
| Alimentation | Fournit l'énergie électrique à tous les composants. |
| Système de refroidissement | Dissipe la chaleur générée par le processeur et les autres composants. |
