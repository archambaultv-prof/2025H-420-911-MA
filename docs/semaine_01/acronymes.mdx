# Sigles et acronymes

Des fois, c'est à en perdre son latin. Voici quelques sigles et acronymes que vous
pourriez rencontrer dans le cours ou dans la nature.

| Acronyme | Signification | Description simple |
|----------|---------------|------------------|
| BIOS     | Basic Input/Output System | Micrologiciel assurant le démarrage de l’ordinateur. |
| CPU      | Central Processing Unit | Effectue les calculs et exécute les instructions. Également le nom du bus pour cette RAM. |
| DDR    | Double Data Rate | Type de RAM permettant un transfert de données plus rapide. |
| DIMM   | Dual In-line Memory Module | Type de connecteur de RAM. |
| DMI   | Direct Media Interface | Bus reliant le CPU au chipset sur les cartes mères modernes. |
| EEPROM| Electrically Erasable Programmable Read-Only Memory | Mémoire non-volatile programmable et effaçable électriquement. |
| FSB     | Front Side Bus | Ancien bus reliant le CPU au chipset (techniquement Northbridge sur les anciennes architectures). |
| GPU      | Graphics Processing Unit | Gère les calculs et l'affichage graphique. |
| HDD      | Hard Disk Drive | Disque dur traditionnel à plateaux magnétiques. |
| HDMI     | High-Definition Multimedia Interface | Connecteur pour audio et vidéo haute définition. |
| I2C    | Inter-Integrated Circuit | Bus série pour communication entre composants sur une carte mère. |
| M.2      | M.2 Interface | Connecteur pour SSD et cartes d'extension sur carte mère. |
| NVMe    | Non-Volatile Memory Express | Protocole de communication pour SSD utilisant PCIe. |
| PCI     | Peripheral Component Interconnect | Ancien bus parallèle pour connecter des périphériques internes. |
| PCIe    | Peripheral Component Interconnect Express | Bus série rapide reliant CPU/chipset à des périphériques internes (GPU, SSD). |
| PCIe x16 | Peripheral Component Interconnect Express x16 | Connecteur PCIe avec 16 lignes pour GPU et autres périphériques. |
| PSU      | Power Supply Unit | Fournit l'alimentation électrique aux composants internes. |
| RAM      | Random Access Memory | Mémoire vive temporaire utilisée par le CPU. |
| RJ45     | Registered Jack 45 | Connecteur pour réseaux Ethernet. |
| SATA     | Serial Advanced Technology Attachment | Interface de connexion pour disques durs et SSD internes. Désigne à la fois le connecteur et l'interface. |
| SPI    | Serial Peripheral Interface | Bus série pour communication entre microcontrôleurs et périphériques. |
| SSD      | Solid State Drive | Disque de stockage rapide, sans pièces mobiles. |
| USB      | Universal Serial Bus | Interface universelle pour périphériques externes. |
| USB-A | Universal Serial Bus Type-A | Connecteur USB standard, non réversible. |
| USB-C | Universal Serial Bus Type-C | Connecteur réversible pour données et alimentation. |
| VGA      | Video Graphics Array | Connecteur vidéo pour moniteurs et projecteurs. |
| Wi-Fi    | Wireless Fidelity | Connexion sans fil à Internet. |
