# Windows Defender

## Introduction

Windows Defender est la solution antivirus et antimalware intégrée à Windows.
Elle est activée par défaut et assure une protection en temps réel contre les
menaces : virus, logiciels espions, ransomwares, etc. Windows Defender
fonctionne de manière fluide avec Windows Update pour recevoir les
dernières définitions de virus.

Depuis Windows 10, Windows Defender fait partie d’un ensemble plus vaste appelé
Sécurité Windows (Windows Security Center), offrant une protection
complète du système.


## Composantes principales de Windows Defender

| Composante                                     | Description                                                                              |
| ---------------------------------------------- | ---------------------------------------------------------------------------------------- |
| **Protection contre les virus et menaces**     | Analyse des fichiers, détection de virus, quarantaine                                    |
| **Protection du compte**                       | Surveillance des connexions et des identités                                             |
| **Pare-feu et protection réseau**              | Intégration avec le pare-feu Windows                                                     |
| **Contrôle des applications et du navigateur** | Protection contre les applications potentiellement dangereuses et les sites web suspects |
| **Sécurité des appareils**                     | Surveillance de l'intégrité matérielle et du démarrage sécurisé                          |
| **Performances et état de l’appareil**         | Rapports sur la santé du système                                                         |
| **Options de famille**                         | Contrôle parental et gestion des comptes enfants                                         |


## Modes d’analyse disponibles

1. **Analyse rapide** : vérifie les emplacements critiques (processus en cours, registre, fichiers de démarrage).
2. **Analyse complète** : vérifie tout le système.
3. **Analyse personnalisée** : permet de sélectionner des dossiers spécifiques.
4. **Analyse hors ligne Windows Defender** : redémarre l’ordinateur et lance une analyse en dehors de Windows pour détecter les malwares profondément enfouis.




## Interface utilisateur

Accessible via :

```
Paramètres → Mise à jour et sécurité → Sécurité Windows
```

Interface graphique pour :

* Voir l’état de la protection
* Lancer une analyse
* Gérer les quarantaines
* Configurer les protections

## Bonnes pratiques

* Ne jamais désactiver Defender sauf pour test ou dépannage spécifique.
* Garder le système à jour pour les dernières signatures de menace.
