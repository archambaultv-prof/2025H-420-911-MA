# Configuration de Windows 11

Voici le compte rendu de l'exercice de configuration de Windows 11. Cette liste
regroupe l'ensemble des configurations mentionnées lors de l'exercice. C'est à
vous de choisir celles qui vous conviennent le mieux.


## Acceuil

- Renommer le poste de travail si besoin.

## Système

### Écran

- Configurer son 2e écran.
- Éclairage nocturne.

### Notifications
- Activer ou désactiver.
- Mode ne pas déranger (planifié ou non).
- Notifications de priorité.

### Concentration
- Activer le mode concentration lorsque vous travaillez sur un projet.

### Alimentation
- Écran et veille : configurer le temps selon vos préférences.

### Storage
- Activer Intelli-stockage.
    - Vider la corbeille après 30 jours.
- Fichier temporaire : configurer le temps selon vos préférences.


### Pour les développeurs
- Activer le mode développeur.
- Explorateur de fichiers :
    - Afficher les extensions de fichiers.
    - Afficher les fichiers cachés.
    - Afficher le chemin complet.
- Permettre l'utilisation de PowerShell pour le scriptage.

### Bureau à distance

- Activer le bureau à distance ou demander aux services informatiques de le faire.

## Option de confidentialité

- Modifier les paramètres de confidentialité selon vos préférences.

## Power Mode
- Power mode : performance.

## Multitasking
- Snap windows selon vos préférences.
- Shake to minimize all others.

## Imprimante
- Configurer les imprimantes.

## Application
- Démarrage par défaut

## Clavier
- Langue de saisie
- Ajouter un clavier US pour coder.

## Compte Microsoft
- Ajouter un compte Microsoft.

## Windows Update
- Mettre à jour Windows.
- Activez informer moi lorsque le redémarrage est nécessaire.
