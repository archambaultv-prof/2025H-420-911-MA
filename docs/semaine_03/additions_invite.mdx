# Additions invité VirtualBox

VirtualBox propose des outils appelés "Additions invité" qui permettent
d'améliorer l'intégration entre l'hôte et la machine virtuelle. Ces outils
permettent notamment de partager le presse-papiers, de redimensionner la
fenêtre de la machine virtuelle, de partager des dossiers entre l'hôte et la
machine virtuelle, etc.

L'installation des Additions invité est recommandée pour une meilleure
expérience utilisateur. Vous pouvez l'installer en deux étapes faciles !

1. D'abord, une fois votre machine virtuelle démarrée, allez dans le menu
"Périphériques" de VirtualBox et sélectionnez "Insérer l'image CD des Additions
invité". Cela montera un CD virtuel dans votre machine virtuelle.
    ![image](/img/VirtualBox/Outil_Additions_invite-1.png)	

2. Ensuite, ouvrez l'explorateur de fichiers dans votre machine virtuelle et
accédez au CD virtuel. Vous y trouverez un fichier exécutable. Double-cliquez
sur ce fichier pour lancer l'installation des Additions invité.
