# Consigne pour l'examen final

- **Dates** : Mercredi 9 juillet 2025 de 12h10 a 15h00.
- **Lieu** : En ligne, via Teams
- **Matière** : Toutes les notions vues en cours et dans les travaux pratiques.
- **Déroulement**: L'examen se déroulera en deux parties:
    - **Questions à choix multiples** : Questionnaire Microsoft Forms
      avec des questions à choix multiples. Vous aurez 45 minutes pour répondre à

    - **Travail pratique** : Après les questions à choix multiples, vous

      sera disponible sur le site web du cours au moment de l'examen.

## Exemples de questions à choix multiples

- La rôle de la carte mère dans un ordinateur est de :
  - [ ] Gérer les connexions réseau
  - [x] Connecter tous les composants matériels
  - [ ] Exécuter des applications
  - [ ] Stocker les données

- Le rôle du BIOS est de :
  - [x] Initialiser le matériel au démarrage
  - [ ] Gérer les connexions réseau
  - [ ] Configurer la carte mère
  - [ ] Le BIOS sont les règlages Windows affectant le démarrage de celui-ci

- Quelle partie de la mémoire partagent les threads d'un même processus ?
  - [x] Le tas
  - [ ] La pile

- Que se passe-t-il si un processus tente d'accéder à une zone de mémoire
  qui ne lui est pas allouée ?
  - [ ] Le processus continue normalement
  - [x] Une erreur de segmentation se produit
  - [ ] Le système d'exploitation redémarre
  - [ ] Le système d'exploitation demande à l'utilisateur s'il veut terminer le processus

- De façon général, comment un ordinateur obtient-il son adresse IP ?
  - [ ] En utilisant le protocole TCP
  - [x] En utilisant le protocole DHCP
  - [ ] En utilisant le protocole FTP
  - [ ] En utilisant le protocole HTTP

## Exemples de travail pratique

- Écrire un script PowerShell pour manipuler des fichiers.
- Écrire un script PowerShell qui collecte des informations sur le système
  d'exploitation pour le support technique. Par exemple :
  - Nom de l'ordinateur
  - Version de Windows
  - Adresse IP
  - Liste des processus actifs
- Écrire un script PowerShell qui vérifie les permissions d'un fichier
  spécifique et affiche un message si l'utilisateur n'a pas les permissions
  nécessaires.

- **Question à choix multiples** : 50 %. La note est automatique calculée par
  Microsoft Forms.
- **Travail pratique** : 50 %.
  - Le script s'exécute sans erreur
  - Le script répond correctement à l'énoncé
  - Le script est bien structuré et lisible
