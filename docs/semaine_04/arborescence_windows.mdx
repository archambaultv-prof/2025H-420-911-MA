# Arborescence des dossiers Windows

Sous Windows, les fichiers et dossiers sont organisés de manière hiérarchique
dans une structure d'arborescence. Chaque élément de cette arborescence est
représenté par un chemin d'accès qui indique sa position dans la hiérarchie.

La racine de l'arborescence est généralement représentée par une lettre de
lecteur (par exemple, C:\ pour le disque dur principal). À partir de cette
racine, les dossiers et fichiers sont organisés en sous-dossiers et fichiers
dans une structure arborescente.

Ainsi, contrairement à Linux, il n'y a pas de répertoire racine unique ("/") mais
un ensemble de répertoires racines pour chaque lecteur. Par exemple, le lecteur
C: a sa propre arborescence, distincte de celle du lecteur D:.
