# Machine virtuelle Windows

Pour tester certaines fonctionnalités de Windows, il peut être utile d'utiliser une machine virtuelle.
Cela va éviter de modifier, voir de casser, votre installation principale de Windows. Voici un
petit guide, très incomplet, pour vous aider à créer une machine virtuelle Windows.

## Prérequis

- Avoir [VirtualBox](https://www.virtualbox.org/wiki/Downloads) installé sur votre machine. Vous pouvez
 aussi utiliser les produits VMware, comme [VMware Workstation](https://www.vmware.com/products/desktop-hypervisor/workstation-and-fusion)
- Télécharger une image ISO de Windows 11. Vous pouvez le faire depuis le [site de Microsoft](https://www.microsoft.com/fr-ca/software-download/windows11).
  Il faut avoir un compte microsoft

## Création de la machine virtuelle

- Dans VirtualBox ou VMware, cliquez sur "Nouvelle" pour créer une nouvelle
  machine virtuelle. Choisissez le mode d'installation standard.

:::warning

L'installation de Windows prend beaucoup de temps. Les hyperviseurs (VirtualBox, VMware, etc.)
sont parfois capricieux. Donc prévoyez faire l'installation plus tôt que tard.

:::
